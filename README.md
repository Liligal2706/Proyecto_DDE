# Dashboard: Redes sociales vs productividad

Este dashboard analiza la relación entre uso de redes sociales y productividad mediante:

- Análisis exploratorio.
- Muestreo estratificado proporcional.
- Estimación puntual e intervalos de confianza.
- Estimadores auxiliares de razón y regresión.
- Diseño en Bloques Completos Aleatorizados.
- ANOVA.
- Verificación de supuestos.
- Comparaciones múltiples.
- Análisis de potencia.
- Reflexión sobre uso de IA.
